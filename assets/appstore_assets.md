Share Hopper（シェアホッパー）は、スポーツを頑張るご家庭を支える、保護者のための成長支援アプリです。

「言葉でアドバイスしてもうまく伝わらない」「過去のフォームと比べてどれくらい上達したか分からない」——そんな"もどかしさ"を、動画とテクノロジーで解消します。

■ 主な機能
───────────────────────────────

・動画のタイムラインに合わせて、任意の時間帯にピンポイントでコメント。
・後から家族で一緒に振り返る際の会話のきっかけになります。
・動画に対するリアルタイムチャットで、遠方の家族やコーチとも状況を共有。

【一人で悩まない、仲間とつながる】
・他の保護者やコーチと動画をシェアし、練習のコツを共有。
・一人で抱え込まず、みんなで日々の成長を支えられます。

───────────────────────────────

───────────────────────────────
・「もっとうまくアドバイスしてあげたい」と感じている保護者様

・言葉だけでなく、動画を使って視覚的にフォームの指導を行いたい方
・家族やコーチと日々の練習状況を共有したい方

───────────────────────────────

## 5. スクリーンショット構成案

App Store では **6.9インチ（iPhone 16 Pro Max）** と **6.5インチ（iPhone 14 Plus等）** の2サイズが必須です。  
iPad 対応する場合は **12.9インチ** も追加します。

### 推奨スクリーンショット順序（5〜10枚）

| # | 画面 | キャプションテキスト例 |
|---|------|----------------------|
| 1 | **ホーム / スペース一覧** | 「動画で残す、成長のアルバム」 |
| 2 | **動画プレーヤー（図形描画オーバーレイ）** | 「ここが違う、を画面で指して伝える」 |
| 3 | **タイムラインコメント表示** | 「気になるシーンに、秒単位でコメント」 |
| 4 | **チャット画面** | 「家族や仲間とつながって成長を支える」 |
| 5 | **スペース招待フロー** | 「メールで簡単に家族・コーチを招待」 |

### スクリーンショット取得コマンド例（Simulator）
```bash
# iPhone 16 Pro Max (6.9インチ)
xcrun simctl io booted screenshot screenshot_01_spaces.png

# シミュレーター起動
open -a Simulator
```

> **推奨ツール**: [Fastlane snapshot](https://docs.fastlane.tools/actions/snapshot/) または [Xcode UI Test + XCUIScreen](https://developer.apple.com/documentation/xctest) でシミュレーターから自動取得。

### デザインガイドライン
- 文字・UI 要素はスクリーンショット上部 30% 以内に配置（ホームボタンなし端末の下部ノッチを考慮）
- 背景色: アプリのブランドカラーに合わせたグラデーションまたは単色
- フォント: SF Pro Display Bold（日本語: ヒラギノ角ゴシック W6）
- キャプションは画像の上または下に重ねてデザイン

## 8. App Store Connect へのビルドアップロード手順

```bash
# 1. Archive ビルド（Xcode）
# Product > Archive または:
xcodebuild archive \
  -project PyutonRemoteAdvisor.xcodeproj \
  -scheme PyutonRemoteAdvisor \
  -configuration Release \
  -archivePath build/PyutonRemoteAdvisor.xcarchive

# 2. IPA エクスポート（ExportOptions.plist が必要）
xcodebuild -exportArchive \
  -archivePath build/PyutonRemoteAdvisor.xcarchive \
  -exportPath build/export \
  -exportOptionsPlist ExportOptions.plist

# 3. App Store Connect へアップロード（altool / Transporter）
xcrun altool --upload-app \
  -t ios \
  -f build/export/PyutonRemoteAdvisor.ipa \
  --apiKey <API_KEY_ID> \
  --apiIssuer <ISSUER_ID>
```

> `ExportOptions.plist` の `method` は `app-store` に設定してください。
